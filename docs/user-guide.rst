User Guide
==========

Task Dashboard aggregates tasks from multiple services into a single unified view.

Perspectives
------------

The dashboard offers four built-in perspectives accessible from the sidebar:

- **My Tasks** — Tasks assigned to you across all connected services.
- **Open Tasks** — All tasks with an open/pending status that you have permission to see.
- **Unassigned Tasks** — Tasks with no owner assigned.

Filtering
---------

Use the filter panel to narrow results by:

- **Search** — Full-text search across title, customer, owner, and group fields.
- **State** — Filter by open, pending, or closed status.
- **Origin** — Filter by service (Zammad, GitLab, EspoCRM, OpenProject, Eramba).
- **Owner** — Filter by assigned user.
- **Group / Service Group** — Filter by team or project group.
- **Priority** — Filter by priority level.
- **Due date** — Filter by due date range using the date picker.

Saved Views
-----------

Save your current filter combination as a named view for quick access later.
Saved views appear in the sidebar and can be deleted when no longer needed.

Refreshing Data
---------------

Tasks are fetched from external services every 5 minutes in the background.
To force an immediate refresh, click the refresh button on a specific task.
